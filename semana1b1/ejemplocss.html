Selectores en css

Un selector en css puede ser:

1-La etiqueta del elemento

2-El id del elemento

3-La clase del elemento

Estos son selectores simples

Pero también, pueden ser una combinación de ellos, por ejemplo:

1-Etiqueta con etiqueta

2-Id con id

3-Class con class

4-O una combinación de cualquiera

Estos son selectores combinados

Tenemos también los selectores por el tipo de atributo
